# Week 1: Hello World & Basic I/O

**Difficulty:** Beginner

## Learning Goals
- Understand how to print output to the console
- Learn how to get input from the user
- Practice using variables and basic string formatting
- Understand basic Python data types (str, int, float)

## Project Description
Build a simple greeting program that asks the user for their name and age, then
prints a personalised greeting with some calculated facts (e.g. birth year).

## Concepts Covered
- `print()` and `input()`
- Variables and assignment
- String concatenation and f-strings
- Type conversion (`int()`, `str()`)
- Basic arithmetic

## How to Run
```bash
python main.py
```

## Sample Output
```
What is your name? Alice
How old are you? 30
Hello, Alice! You are 30 years old.
You were born around the year 1996.
In 10 years you will be 40 years old.
```

## Challenges
1. Add a line that prints how many months old the user is.
2. Ask the user for their favourite colour and include it in the greeting.
3. Validate that the age entered is a positive number.
